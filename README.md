# 🍎 SIA Penjualan Buah

Website sederhana berbasis PHP & MySQL untuk mengelola penjualan buah.

## 🚀 Fitur
- Tambah produk
- Upload gambar
- Tampilkan data produk
- Database otomatis

## 🛠️ Teknologi
- PHP Native
- MySQL
- CSS

## ▶️ Cara Menjalankan
1. Install XAMPP
2. Copy folder ke htdocs
3. Jalankan Apache & MySQL
4. Buka:
   http://localhost/sia-buah

## 📁 Struktur Folder
- config → koneksi database
- pages → halaman
- proses → logic
- assets → css & gambar
